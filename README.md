# AUTONUMY Proof of Reserves Attestations

Public, auditable record of Proof of Reserves attestations generated by [AUTONUMY](https://autonumy.com).

Each attestation contains:
- **Merkle Root** — SHA-256 root of the liability tree
- **Coverage Ratio** — total assets / total liabilities
- **Snapshot Hash** — integrity hash of the underlying data
- **Schema Version** — attestation format version

## Verification

Each tagged commit contains the full attestation JSON. The Merkle root can be independently verified against the on-chain record (when Polygon attestation is enabled).
